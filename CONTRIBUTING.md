# Contributing to 0p3n-s3s@m3

Thank you for helping build this quick-reference security toolkit! We welcome contributions from the community.

## How to Contribute
1. **Fork the Repository**: Create your own copy of this project to work on.
2. **Add Your Reference Material**:
   * Keep cheat sheets clear, concise, and scannable.
   * Provide examples for realistic, common use cases.
   * Format any code or commands properly in markdown syntax blockquotes.
3. **Submit a Pull Request**: Explain exactly what you added or improved in your PR description.

## Content Guidelines
* Ensure all scripts and commands provided focus purely on educational resources and authorized penetration testing techniques.
* Keep explanations clear for beginners—just like our "Day 01" introduction.
